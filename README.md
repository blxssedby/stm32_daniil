# STM32 Daniil Project

STM32 прошивка с анимированным выводом "Daniil" через UART.

## Описание
Проект демонстрирует работу с регистрами STM32F103 и UART передачей данных.
Выводит 4 разных стиля текста "Daniil" в циклическом режиме.

## Стили вывода
- DANIIL
- ** Daniil **
- ==> Daniil <==
- -D-A-N-I-I-L-

pio run
